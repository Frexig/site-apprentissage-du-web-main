Concepts clés

Partie 1 - Découvrez le web et ses origines

Web

Définition : Système d'information qui permet d'accéder à des contenus via internet.

Internet

Définition : Réseau qui permet de communiquer entre ordinateurs.

Cloud

Définition : Ensemble des serveurs distants proposant des services accessibles par le réseau internet.

W3C

Définition : Organisation regroupant plusieurs acteurs du web avec pour but la création et la mise à jour de protocoles et technologies permettant d'assurer le développement de la toile sur le long terme.

Partie 2 - Distinguez les langages du Web

Langages client

Définition : Le langage client sert à fournir des informations de base pour la construction des sites web.

Navigateurs

Définition : Logiciel qui permet de consulter les pages internet.

Langages serveur

Définition : Un langage serveur décrit comment un site web doit se comporter. Ce langage de programmation est exécuté sur un serveur, seul le résultat de l'éxécution est envoyé à l'ordinateur client.

Frameworks

Définition : Ensemble de bibliothèques, outils, de conventions de codage et de modèles de conception qui simplifient et accélèrent le processus de développement logiciel.

Bases de données

Définition : Bibliothèque digitalisée utilisée pour stockker et organiser les données afin qu'elles soient plus faciles à gérer et à consulter.

Site responsive

Définition : Approche de conception web visant à modifier l'apparence d'un site selon la taille de l'écran et le support utilisé par l'internaute.

Application mobile

Définition : Programme téléchargeable sur smartphone ou tablette qui comporte un fichier qui est installé puis exécuté par le système d'exploitation de son mobile.

Partie 3 - Découvrez le fonctionnement du réseau

Serveurs

Définition : Ordinateur qui exécute des opérations suivant les requêtes effectuées par un autre ordinateur appelé « client ».

Adresse IP

Définition : Représentation numérique de l'endroit où un appareil est connecté à Internet.

Nom d’hôte et DNS

Définition : Nom d'hôte : Nom d'un périphérique dans un réseau. Le DNS (Domaine Name System) permet de traduire le nom d’hôte en adresse IP.

Protocoles

Définition : Norme ou un ensemble de règles qui définit comment les ordinateurs communiquent entre eux.

Partie 4 - Identifiez les métiers du web

Développeurs

Définition : Technicien en charge de la réalisation d'un système ou d'une application

Product Managers et chefs de projets

Définition : Product Manager : Le Product Manager construit la vision stratégique d'un produit, interprète et anticipe les besoins des clients et des prospects. Chef de projet : Personne qui organise et conduit les différentes phases du projet dont il en assume entièrement le management.

Designers

Définition :Il crée des produits (sites web, logiciels, applications mobiles, objets connectés ou interactifs, etc.) ou services multimédias en y intégrant des sons, du texte, des images, en étroite collaboration avec une équipe aux compétences transversales.

Testeurs

Définition : Professionnel spécialisé en informatique chargé de réaliser des tests et des vérifications sur différents logiciels afin de garantir leur bon fonctionnement.

Techniciens informatique

Définition : Il veille à l’intégrité du système informatique et prend en charge les installations, la maintenance, les incidents et le support utilisateur.

Langages de développement web

HTML

Front-End

Squelette du site Web. Il définit la structure de base du contenu, comme les titres, les paragraphes, les images, etc.

CSS

Front-End

Définition : Ensemble des éléments visibles et accessibles directement sur un site web.

JavaScript

Front-End

Définition : Ensemble des éléments visibles et accessibles directement sur un site web.

JavaScript

Back-End

Définition : JavaScript est un langage de programmation utilisé par les développeurs pour concevoir des sites web interactifs. Les fonctions JavaScript peuvent permettre d'améliorer l'expérience utilisateur d'un site web, de la mise à jour des flux de médias sociaux à l'affichage d'animations et de cartes interactives.

PHP

Back-End

Définition : Langage de programmation libre. LE PHP est un langage interprété, cela veut dire que celui-ci doit être interprété par un serveur afin d'afficher correctement le contenu demandé.

Java

Back-End

Définition : Java est un langage de programmation largement utilisé pour coder des applications web. Il s'agit d'un langage de programmation rapide, sécurisé et fiable qui permet de tout coder, des applications mobiles aux logiciels d'entreprise en passant par les applications de big data et les technologies côté serveur.

Python

Back-End

Définition : Python est un langage de programmation largement utilisé dans les applications Web, le développement de logiciels, la science des données et le machine learning (ML).

C#

Back-End

Définition : Langage de programmation à objets, qui combine des éléments provenant essentiellement de C/C++ et de Java.

SQL

Bases de données

Définition : Bibliothèque digitalisée utilisée pour stocker et organiser les données afin qu'elles soient plus faciles à gérer et à consulter.

Boîte à outils du développeur

  • IDE comme VS Code : Définition : Application logicielle qui permet aux développeurs de créer, d'éditer et de compiler du code au sein d'un même programme.
  • Github : Définition : Plateforme open source de gestion de versions et de collaboration destinée aux développeurs de logiciels.
  • Librairies / bibliothèques : Définition : ensemble de fonctions, de classes ou de modules pré-écrits et réutilisables qui peuvent être intégrés dans un programme pour effectuer des tâches spécifiques.
  • Frameworks : Définition + 3 exemples avec le langage associé : Un framework est un ensemble de bibliothèques, outils, de conventions de codage et de modèles de conception qui simplifient et accélèrent le processus de développement logiciel. Exemples : PHP = Symfony; JAVA = Java EE; PYTHON = Django
